Resource Directory (RDDL) for the Darwin Core Curatorial Extension

Overview

This is the root namespace of the Darwin Core Curatorial extension For all resources related to Darwin Core please see http://rs.tdwg.org/dwc.

Note: While this document is human-readable, RDDL-aware processors may use it to automatically retrieve resources such as different schema versions for validating documents. See http://www.rddl.org for further information.

The root (target) namespace of the Darwin Core Curatorial extension is: http://rs.tdwg.org/dwc/curatorial.
