1. Introduction

The Darwin Core is meant to serve a global community. Access to the documents describing the standard in multiple languages will facilitate open access and global adoption. This page is meant to be an index to all translations of Darwin Core documents. Anyone interested in providing translations to be hosted with the corpus of the standard should contact the Darwin Core Task Group [DWC-TASK].
